# Rule: Government-Grade Accessibility

1. **Mandatory Libraries:** You must exclusively use Tailwind CSS and `shadcn/ui` for all component styling.
2. **No Custom CSS:** Avoid writing custom CSS classes unless absolutely necessary to override a library default.
3. **Accessibility Standard:** The interface must comply with government web accessibility standards, ensuring high contrast, screen-reader compatibility, and easy navigation for semi-literate citizens.
