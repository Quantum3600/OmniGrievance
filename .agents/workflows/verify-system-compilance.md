# Workflow: System Verification Protocol

1. **Verify Ingestion:** Trigger the `/intake` endpoint with a mock WhatsApp payload. [cite_start]Check if the AI Gateway correctly normalizes the JSON[cite: 86, 87].
2. [cite_start]**Verify Triage:** Check the database to see if the PostGIS spatial query correctly assigned the ticket to a Nodal Officer based on coordinates[cite: 91].
3. [cite_start]**Verify RBAC & CRM:** Log in as a 'Government Employee' and ensure the dashboard only displays tasks within your jurisdiction[cite: 36, 94].
4. [cite_start]**Verify Gamification:** Resolve a task with photographic proof and verify that 'Reward Points' are incremented in the user profile[cite: 95, 116].
5. [cite_start]**Verify Transparency:** Check the Public Portal to ensure the resolved ticket and proof photo are visible[cite: 49, 97].
